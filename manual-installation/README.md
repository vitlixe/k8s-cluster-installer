# Ручная установка Kubernetes

Подробные пошаговые инструкции для установки Kubernetes кластера вручную.

## Файлы

### master-node-setup.md
Полная инструкция по установке и настройке master ноды.

**Что будет установлено:**
- Container runtime (containerd)
- Kubernetes компоненты (kubelet, kubeadm, kubectl)
- Инициализация кластера
- Calico CNI
- Настройка kubectl

**Что вы узнаете:**
- Как работает SWAP и почему его нужно отключить
- Какие модули ядра нужны для Kubernetes
- Как настроить containerd с systemd cgroups
- Как работает kubeadm init
- Как устроен Calico CNI

### worker-node-setup.md
Полная инструкция по установке и подключению worker ноды к кластеру.

**Что будет установлено:**
- Container runtime (containerd)
- Kubernetes компоненты (kubelet, kubeadm, kubectl)
- Присоединение к кластеру

**Что вы узнаете:**
- Как работает команда kubeadm join
- Как Worker нода подключается к Master
- Как Calico настраивает сеть на новой ноде

## Порядок установки

1. **Сначала Master нода** - управляет кластером
2. **Затем Worker ноды** - исполняют рабочую нагрузку
3. **Повторить для каждой Worker ноды** - можно добавлять сколько угодно

## Альтернатива

Для быстрой установки без деталей используйте автоматическую установку.
